 26  """ 
 27  P4VASP is a library for processing outputs from a VASP program 
 28  (http://cms.mpi.univie.ac.at/vasp/Welcome.html) 
 29  It is focussed mainly on the new xml output format, 
 30  but it offers also limited support for some older formats 
 31  (POSCAR, INCAR). 
 32   
 33  P4Vasp contains also an extensible GUI for visualisation and 
 34  an api for (reasonably) easy creation of extension applets. 
 35   
 36  For people, who want to write just a simple script rather than 
 37  an real applet, the easiest way 
 38  to get informations from the *vasprun.xml* file is to use 
 39  the XMLSystemPM class defined in p4vasp.SystemPM.
