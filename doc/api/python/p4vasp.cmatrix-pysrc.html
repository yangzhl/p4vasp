 30  """ 
 31  Basic *Vector* and *Matrix* manipulation library. 
 32  This module defines objects *Vector* and *Matrix* with (almost) 
 33  the same api as the matrix.py, but cmatrix.py uses functions written in C. 
 34  The other difference is, that it is restricted to 3d vectors and matrices only. 
 35   
 36  There are two reasons to use this module: 
 37   
 38    * performance - the performance of *cmatrix* can be from 1.5 up to more than 10 times 
 39      better than for the *matrix* (depending on the operation) 
 40   
 41    * *cmatrix* can provide a convenient interface to other 3d vector-based 
 42      objects implemented in C. 
 43   
 44  Objects *Matrix* and *Vector* are instances of the corresponding objects 
 45  from the *matrix* module. 
 46  Objects of *cmatrix* can interact with the *matrix* objects - mixed types 
 47  can be used in operations. 
 48   
 49  Limitations: 
 50   
 51    * only 3d objects, 
 52   
 53    * only *float* (double) type. 
 54   
 55  Bugs: Sometimes the objects needs to be deleted explicitely by *del* statement 
 56  in order to avoid exceptions. 
 57  """
