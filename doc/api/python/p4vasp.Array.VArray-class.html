Class VArray

source code

One dimensional array of vectors (i.e. one-dimensional arrays). All elements of VArray must have the same type ( kept in variable *type* ).

Variable *name* contains the name attribute and *element* (optionaly) contains the element from which the VArray was constructed.

__init__(self, data=[], t='float', name=None, fastflag=0)

source code 

Create the VArray. *data* can be a python list, *VArray*, *UserList* or *numpy* array instance, or XML element node.

*name* can specify the name attribute. *t* can specify the type (FLOAT_TYPE by default). *fastflag* if true, *readFromNodeFast* is used to read from *data*

readFromNodeFast(self, data)

source code 

Read the vArray content from the XML element node. Old data are replaced by the new content. The content is parsed in C++, using *readFromNode* as a fallback. This procedure can be used for reading vArray of float 3d vectors only. The vectors are stored as cmatrix Vectors.
