Class PropertyManager

source code

PropertyManager is a dictionary of properties. It works as a dictionary of Property instances, but the properties are also available as attributes. The simplest way how to obtain property X from PropertyManager pm is to call value=pm.X, which is equivalent to value=pm["X"].get()
