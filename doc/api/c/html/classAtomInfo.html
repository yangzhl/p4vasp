Detailed Description

Array of AtomtypesRecords.

This class can hold several AtomtypesRecord structures and provides access to them.

There are reserved Atominfo::allocated AtomtypesRecords in the memory (accessible through the AtomInfo::atomtypes pointer). However, the actual array can be smaller. Its length is kept in the AtomInfo::types variable. The array-elements can be accessed through various functions (AtomInfo::getRecord(int), operator []) which (can) perform index range checking. New record(s) can be added using AtomInfo::append(AtomtypesRecord*) and AtomInfo::allocate(int).

◆ realloc()

void AtomInfo::realloc ( int  alloc)

Reserve memory for alloc AtomtypesRecords.

If alloc==allocated, nothing happens If alloc>types, the memory is reallocated and old AtomtypesRecords are copied. If alloc<types, the memory is reallocated and first alloc old AtomtypesRecords are copied, AtomInfo::types is set to alloc. If alloc==0, memory is cleaned, AtomInfo::types and AtomInfo::allocated is set to zero and AtomInfo::atomtypes is set to NULL.

int AtomInfo::allocation_step

Allocation step.

When more records are required than are available (AtomInfo::allocated), e.g. by calling AtomInfo::append(AtomtypesRecord*), then at least allocated+allocation_step AtomtypesRecords are allocated.
